- **Yearly:** GitHub-style daily grid (52 weeks x 7 days)
- **Monthly:** 4-5 weeks detailed view
- **Weekly:** 7-day detailed view with completion counts

### Graph Intensity
- Darker shades = higher completion rate
- For multi-daily habits: intensity based on % of target met
- For weekly habits: shows weeks instead of days

### Interactions
- Scroll vertically through habits
- Dropdown to change time frame (Yearly/Monthly/Weekly)
- Tap on a cell to see details for that day/week

### Reminders
- Multiple reminders per habit supported
- Each reminder has a time picker and enable/disable toggle (â—‹/â—)
- Reminders only fire on days when the habit is scheduled
- [Ã—] button to delete a reminder

### Frequency Options Detail
1. **Daily:** Every single day, once per day
2. **Multiple times per day:** e.g., drink water 8x/day
3. **Specific days:** e.g., Mon/Wed/Fri, with optional times per day
4. **Every N days:** e.g., every 3 days
5. **Weekly:** Once per week (shows in weekly graph view)
